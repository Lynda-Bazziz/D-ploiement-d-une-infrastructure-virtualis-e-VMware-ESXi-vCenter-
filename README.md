# Déploiement d'une infrastructure virtualisée VMware-ESXi-vCenter
Ce projet consiste à déployer un laboratoire VMware vSphere 7.0 comprenant deux hyperviseurs ESXi, un serveur vCenter Server Appliance, un stockage partagé FreeNAS/TrueNAS via iSCSI et un contrôleur de domaine Windows Server 2019 avec Active Directory et DNS.
L’objectif est de mettre en pratique l’administration d’un environnement virtualisé : configuration réseau (vSwitch/vDS), gestion des datastores VMFS, création et gestion des machines virtuelles, snapshots, clonage, templates, migrations vMotion/Storage vMotion ainsi que la mise en place de fonctionnalités avancées comme la haute disponibilité (HA).

Technologies : VMware ESXi 7.0, vCenter, FreeNAS/TrueNAS, iSCSI, VMFS, Windows Server 2019, Active Directory, DNS, Alpine Linux.
